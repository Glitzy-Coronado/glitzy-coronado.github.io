## 🚀 Deploy to GitHub Pages

1. Create a new GitHub repository (e.g. `glitzy-portfolio`)
2. Upload **all files and folders** maintaining the structure above
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch → `/ (root)`
5. Click **Save**
6. Live at: `https://yourusername.github.io/glitzy-portfolio`

> ⚠️ Keep the folder structure intact. The images folder must be at `assets/images/` relative to `index.html`.
